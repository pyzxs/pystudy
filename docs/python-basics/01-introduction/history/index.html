    Python 的发展历史 #

    Python 的历史可以追溯到 20 世纪 80 年代末。它的创始人 Guido van Rossum 在 1989 年的圣诞节期间，为了打发圣诞节假期时间，决定开发一个新的脚本解释程序，作为 ABC 语言的一种继承。

    重要时间节点 #

    • 1991：第一个 Python 编译器诞生
    • 2000：Python 2.0 发布
    • 2008：Python 3.0 发布
    • 2020：Python 2 停止维护
