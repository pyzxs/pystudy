    一、准备工作 #

    1. 检查是否已安装 Python #

    • Windows：

      python --version

    • macOS/Linux：

      python3 --version

      若显示版本号（如 Python 3.12.1），则已安装。

    2. 下载 Python 安装包 #

    • 官网下载：https://www.python.org/downloads/
    • 选择最新稳定版（如 Python 3.12.x）
    • 注意：避免下载 Python 2.x（已停止维护）

    二、Windows 安装步骤 #

    1. 运行安装程序 #

    双击下载的 .exe 文件（如 python-3.12.1-amd64.exe）

    2. 关键配置选项 #

    • ✅ 勾选 Add python.exe to PATH（必选项！）
    • 选择 Customize installation（自定义安装）

    3. 可选功能（推荐全选） #

    • ✅ pip（Python包管理工具）
    • ✅ py launcher（多版本管理）
    • ✅ for all users（所有用户可用）

    4. 高级选项 #

    • ✅ Install for all users
    • ✅ Precompile standard library
    • 安装路径建议保持默认（C:\Python312\）

    5. 完成安装 #

    点击 Install → 等待完成 → 关闭窗口

    6. 验证安装 #

    打开命令提示符（CMD）：

    python --version  # 应显示版本号
pip --version     # 应显示pip版本

    测试Python交互模式：

    python
>>> print("Hello Python!")
Hello Python!
>>> exit()

    三、macOS 安装步骤 #

    方法一：官方安装包（推荐） #

    1. 下载 .pkg 文件 → 双击运行
    2. 全程点击 继续，无需修改配置
    3. 验证安装：
    python3 --version
pip3 --version

    方法二：Homebrew（开发者推荐） #

    1. 安装Homebrew（如未安装）：
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    1. 通过Brew安装Python：
    brew install python

    四、Linux 安装步骤（以Ubuntu为例） # 

    # 1. 更新系统
sudo apt update && sudo apt upgrade -y

# 2. 安装Python和pip
sudo apt install python3 python3-pip -y

# 3. 验证安装
python3 --version
pip3 --version

# 4. 可选：设置默认Python版本
sudo update-alternatives --config python3

    五、环境变量配置（故障排查） #

    常见问题 #

    • 命令提示符中 python 或 pip 报错"不是内部命令"

    Windows解决方案 #

    1. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
    2. 在 系统变量 中编辑 Path
    3. 添加两条路径： 
      C:\Python312\
C:\Python312\Scripts\
    4. 重启命令提示符

    六、安装后必做事项 #

    1. 升级pip # 

    # Windows
python -m pip install --upgrade pip

# macOS/Linux
python3 -m pip install --upgrade pip

    2. 安装虚拟环境工具 # 

    # Windows
pip install virtualenv

# macOS/Linux
pip3 install virtualenv

    3. 配置IDE示例（VS Code） #

    1. 安装Python扩展
    2. 按 Ctrl+Shift+P → 输入 Python: Select Interpreter
    3. 选择已安装的Python版本

    七、常见问题解决方案 #

    1. 安装失败 #

    • 关闭杀毒软件
    • 以管理员身份运行安装程序
    • 重新下载安装包

    2. 多版本共存 #

    • Windows：使用 py 命令 
      py -3.10  # 启动Python 3.10
py -3.12  # 启动Python 3.12
    • macOS/Linux： 
      python3.10  # 调用Python 3.10

    3. pip安装包速度慢 #

    配置国内镜像源：

    pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

    八、测试你的安装 #

    1. 创建 hello.py 文件，内容为：
    print("Python安装成功！")
print(f"运行版本：{sys.version}")
    1. 终端运行：
    # Windows
python hello.py

# macOS/Linux
python3 hello.py

    恭喜！ 现在可以开始你的Python编程之旅 🐍
    下一步建议：学习基础语法 → 尝试安装常用库（如 pip install numpy pandas）
